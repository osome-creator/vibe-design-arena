# Color Palettes

Curated, production-ready color palettes organized by mood and industry. Each palette includes hex values for primary, secondary, accent, background, surface, text, and muted roles. All palettes are designed to meet WCAG AA contrast requirements (≥4.5:1 for body text).

Use OKLCH for new projects; hex is provided for maximum compatibility.

## How to Use

1. **Match mood to product.** A meditation app doesn't want Flame + Charcoal. A gaming platform doesn't want Navy + Steel.
2. **Check contrast.** Body text must hit ≥4.5:1 on its background. If a palette's muted color is too light, bump it darker.
3. **OKLCH for code.** Use OKLCH values in CSS (`oklch(0.52 0.14 35)`) for perceptually uniform color manipulation — lighten/darken by changing L, saturate by changing C.
4. **Three-color rule.** Most interfaces need only: primary (brand), accent (CTAs, links), and neutral (backgrounds, text, borders). Resist adding a fourth.
